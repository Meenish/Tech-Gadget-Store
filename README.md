# Concepts I Learned

## CSS Variables

```css
:root{
--cyan:#00f0ff;
--text:#f5f7ff;
}
```

Variables allow defining values once and using them multiple times.

Instead of:

```css
color:#00f0ff;
```

use:

```css
color:var(--cyan);
```

This makes code cleaner.

---

## Sticky Navigation

```css
position:sticky;
top:0;
```

Keeps navigation visible while scrolling.

---

## Backdrop Blur

```css
backdrop-filter:blur(18px);
```

Creates glass effect.

---

## Flexbox

```css
display:flex;
```

Used for arranging content in a row or column.

---

## CSS Grid

```css
display:grid;
```

Used for layouts with rows and columns.

---

## Media Queries

```css
@media(max-width:980px)
```

Makes website adapt for mobile devices.

---

## Semantic HTML

Instead of:

```html
<div>
```

Use:

```html
<header>
<nav>
<section>
<main>
<footer>
```

Makes structure cleaner and professional.
